# Champ-Select Coach — Downloads

*[Version française](README.md)*

League of Legends champ-select assistant: automatic picks/bans, AI briefing, champion notes, voice announcements.

**[Download the latest version](https://github.com/lucbruno/champ-select-coach-releases/releases/latest)** — file "Champ-Select Coach Setup x.y.z.exe".

## Installation

1. Download the Setup `.exe` above and run it.
2. Windows SmartScreen may show "Windows protected your PC": click **More info → Run anyway** (the installer is not code-signed).
3. On first launch, open **Settings → Artificial intelligence** and paste your API key (free Gemini key: https://aistudio.google.com → Get API key).
4. For English voice announcements nothing is needed; for French ones, install the Windows French voice (Settings → Time & Language → Language & region → Add French → check Text-to-speech).

The app updates itself automatically whenever a new version is published here.

*This repository only hosts published versions — the source code is private.*

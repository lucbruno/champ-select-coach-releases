# League Herald Pilot — Downloads

*[Version française](README.md)*

League of Legends desktop companion: auto-accept, automatic picks and bans, rune page
selection, and voice announcements (game start, minions, objectives called out before
they spawn — with your own sounds if you want).

**[Download the latest version](https://github.com/lucbruno/league-herald-pilot-releases/releases/latest/download/League-Herald-Pilot-Setup.exe)** — file "League-Herald-Pilot-Setup.exe".

## Installation

1. Download the Setup `.exe` above and run it.
2. Windows SmartScreen may show "Windows protected your PC": click **More info → Run
   anyway** (the installer is not code-signed).
3. For English voice announcements nothing is needed; for French ones, install the
   Windows French voice (Settings → Time & Language → Language & region → Add French →
   check Text-to-speech).

The app updates itself automatically whenever a new version is published here.

*This repository only hosts published versions — the source code is private.*

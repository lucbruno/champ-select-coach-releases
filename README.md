# League Herald Pilot — Téléchargements

*[English version](README.en.md)*

Compagnon de bureau pour League of Legends : accept de file, picks et bans automatiques,
sélection de page de runes, et annonces vocales (début de partie, sbires, objectifs
annoncés avant leur apparition — avec tes propres sons si tu veux).

**[Télécharger la dernière version](https://github.com/lucbruno/league-herald-pilot-releases/releases/latest/download/League-Herald-Pilot-Setup.exe)** — fichier « League-Herald-Pilot-Setup.exe ».

## Installation

1. Télécharge le Setup `.exe` ci-dessus et lance-le.
2. Windows SmartScreen peut afficher « Windows a protégé votre PC » : clique
   **Informations complémentaires → Exécuter quand même** (l'installateur n'est pas signé).
3. Pour les annonces vocales en français : installe la voix Windows française
   (Paramètres → Heure et langue → Langue et région → Ajouter Français → cocher
   Synthèse vocale).

L'application se met à jour automatiquement à chaque nouvelle version publiée ici.

*Ce dépôt ne contient que les versions publiées — le code source est privé.*
